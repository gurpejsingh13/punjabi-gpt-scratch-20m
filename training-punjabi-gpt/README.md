# Data

This project uses Punjabi text data derived from the AI4Bharat Sangraha corpus (Punjabi / Gurmukhi split), followed by local filtering and cleaning.

The full training data is not included in this repository due to size constraints.

Processed corpus summary is available in:
- `outputs/corpus_summary.json`

To reproduce the pipeline, use:
- `notebooks/notebook1_download.ipynb`
- `notebooks/notebook2_corpus_analysis.ipynb`
